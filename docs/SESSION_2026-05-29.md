## エンコーディングルール（確定版）

### SET (sub=0x09) — IEEE 754 float, 5×7bit packed

| ブロック | パラメータ | エンコーディング |
|---------|-----------|----------------|
| Amp/Drive | continuous (Gain, Bass, etc.) | normalized: value/max → 0.0-1.0 |
| Amp/Drive | bipolar (Level, Balance) | raw_float: display value directly |
| Amp/Drive | switch | raw_float: 0.0 or 1.0 |
| Effect blocks | `*_MIX`, `*_LEVEL`, `*_PAN` | **normalized**: value/max → 0.0-1.0 |
| Effect blocks | continuous (Time, Feed, etc.) | raw_float: display value directly |
| Effect blocks | bipolar | raw_float: display value directly |
| Effect blocks | frequency (Hz) | raw_float: Hz value directly |
| Effect blocks | enum | raw_float: integer index |
| Effect blocks | switch | raw_float: 0.0 or 1.0 |
| Effect blocks | signed_int (Pitch Shift) | raw_float: semitone value |
| Cab | DynaCab R/Z | normalized: 0.0-1.0 |
| Cab | everything else | raw_float |

### GET (func=0x1F) — 21bit integer, 3×7bit packed

全ブロック共通: `raw = lo | (hi << 7) | (msb << 14)`, range 0-65534
- continuous: `value = raw / 65534 * max`
- bipolar: `value = raw / 65534 * (max - min) + min`
- switch: `value = raw != 0`
- enum: `value = raw` (integer index)
